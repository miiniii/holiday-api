# Holiday API

공공 API를 활용해 국가별 연도별 공휴일 정보를 저장하고 조회할 수 있는 Spring Boot 기반 REST API입니다.

## 기술 스택

- Java 21
- Spring Boot 3.4.4
- JPA (Hibernate)
- H2
- QueryDSL
- Spring Scheduler
- Swagger
- Gradle



## 실행 방법
```bash
git clone https://github.com/miiniii/holiday-api.git
cd holiday-api
애플리케이션 실행
./gradlew bootRun
```
